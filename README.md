# Description

This is a all-in-one Tool for IT Companys! You can manage Assets, Devices.
Plus you can write and manage MD Documentations!

# Features
And Good:

- Custom managable Dashboard
- Manage Devices, Servers and VMs
- Write and Manage MD Docs
- Write weekly Reports
- More planned!
